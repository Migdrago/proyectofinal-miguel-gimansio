Proyecto de gimnasio.

Base de datos:
Se han creado tablas de clientes, clases, entrenadores e inscripciones con sus relaciones.

Programación:
Aplicación en Java con menú que permite gestionar clientes, clases y entrenadores.

XML:
Se ha creado un XML con los datos del gimnasio.

XSD:
Se ha diseñado un esquema que valida el XML.

Validación:
El XML se ha validado correctamente con el XSD (ver carpeta /docs).

Integración:
El XML representa los mismos datos que la base de datos, por lo que puede usarse como exportación.
